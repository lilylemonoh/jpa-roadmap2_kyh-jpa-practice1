# 참고사항
강의와 다르게 진행한 점:
* 인메모리 모드를 사용하여 H2 데이터베이스를 사용함 (h2 미설치)
* 따라서 application.yml에 다음과 같이 설정함
```
spring:
  datasource:
    url: jdbc:h2:mem:testdb
    username: sa
  h2:
    console:
      enabled: true
```
